# 旅游景点数据分析与查询系统

## 系统简介

这是一个旅游景点数据分析与可视化大屏系统，可以实时展示景点销量排名、价格分布等信息。

## 主要功能

### 1. 实时排名
展示今日销量 TOP10 的热门景点，数据每 0.5 秒自动刷新变化。

### 2. 历史销量排行
展示历史累计销量 TOP10 的景点，按销量降序排列。

### 3. 省份销量分布
中国地图热力图展示各省景点销量汇总，颜色越深表示销量越高。

### 4. 价格分布
气泡图展示景点价格区间分布（0-500元，每50元一档），气泡越大表示该价格区间的景点越多。

### 5. 智能问答
可以用自然语言查询景点信息，例如：
- 上海有哪些景点
- 价格最低的景点是哪个
- 评分最高的景点有哪些
- 哪个景点的销量最高

## 数据字段说明

景点数据包含以下字段：
- 名称：景点名称
- 城市：所在城市
- 星级：景点等级（如4A、5A）
- 评分：用户评分（0-10分）
- 价格：门票价格（元）
- 销量：历史累计销量
- 省/市/区：完整地址
- 坐标：经纬度坐标
- 简介：景点简短介绍
- today_sales：今日实时销量

## 技术架构

### 前端
- HTML/CSS/JavaScript
- ECharts 图表库
- 静态页面部署

### 后端
- FastAPI Python Web 框架
- SQLite 数据库存储景点数据
- DashScope 通义千问 LLM

### 数据流程
1. 后端定时更新景点实时销量（每0.2秒）
2. 前端每0.5秒请求最新排名数据
3. 用户可通过对话页面用自然语言查询景点信息
